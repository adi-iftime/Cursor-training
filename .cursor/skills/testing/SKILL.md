# Skill — Testing

## Purpose

Define and execute automated tests that gate quality for code and data.

## When to use

- Tester for pytest suites; Data Engineer for TDD; Data Quality for rule codification.

## Practices

- Deterministic fixtures; mark integration tests separately.
- Failures must include contract or module identifiers.
