# Skill — Databricks

## Purpose

Build and operate Spark-based pipelines, jobs, and Unity Catalog objects on Databricks.

## When to use

- Data Engineer implements batch/streaming on Spark.
- DevOps deploys job definitions; Cost Optimizer reviews cluster usage.

## Practices

- Use Unity Catalog for governance alignment; tag jobs with `correlation_id`.
- Prefer job clusters or SQL warehouses sized to workload class.
